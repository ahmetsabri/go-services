# Authenticator 

 ### Simple API authentication service written in GO


## What I used

- GORM
- Gorilla mux

You can test it by importing  [go-authenticator.json](https://github.com/ahmetsabri/go-services/blob/main/authenticator/Go-authenticator.json)

## Usage
 - Clone the repo
 - run `go run main.go`
